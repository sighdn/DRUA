The dataset is the France dataset. These images were captured
by Pleiades and WorldView 2, showcasing a road construction
event in Toulouse, France, spanning from May 2012 to July
2013. Both images have a size of 2000 × 2000 pixels with a
spatial resolution of 0.52m.
This dataset was introduced in the following paper：Multimodal Change Detection in Remote Sensing Images Using an Unsupervised Pixel Pairwise-Based Markov Random Field Model
